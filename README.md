⚙️ Local Setup
bash
```
# Clone repository
git clone https://github.com/Naveen-Beniwal/Basic_Projects_0.git

# Install dependencies
npm install

# Database
Ensure MongoDB is running locally
mongodb://127.0.0.1:27017/pinterest

# Start server
npm start

# Access application
Visit http://localhost:3000
```
